Plataforma diseñada bajo el paradigma de Serverless Architecture, utilizando Supabase para persistencia en tiempo real y Groq para procesamiento cognitivo. El sistema garantiza una experiencia fluida al eliminar la carga de procesamiento en el cliente mediante funciones distribuidas en la periferia.

Experiencia de Usuario (UX)

  • Catálogo AI-Driven: Motor de búsqueda semántica y filtros dinámicos.
  • Interfaz Adaptativa: Layout 100% responsivo con componentes optimizados.
  • Frictionless Checkout: Proceso de pago simplificado en 3 pasos.
  • Persistence Engine: Carritos que se sincronizan entre dispositivos.
  • Micro-interacciones: Animaciones que guían al usuario durante la compra.

Inteligencia Artificial

  • Deep Integration: IA entrenada con el catálogo para asesoría real.
  • Acción Directa: Generación de botones de compra dentro del chat.
  • Zero-Latency: Procesamiento optimizado vía Groq Cloud.
  • Personificación: Tono de voz alineado con MYB Hijos del Rey.
  • Smart Prompts: Entendimiento de lenguaje natural y errores ortográficos.

Seguridad y Pasarelas de Pago

  • Tokenización Wompi: Procesamiento de datos bancarios cumpliendo con certificado PCI-DSS.
  • Multi-Canal: Soporte para tarjetas de crédito, PSE, Nequi y transferencias manuales vía WhatsApp.
  • Auth de Grado Bancario: Gestión de identidades mediante Supabase Auth con soporte para MFA.
  • RLS (Row Level Security): Protección de datos directo en la base de datos por cada usuario.

Ecosistema de Administración (Back-Office)

Panel centralizado para el control operativo total del negocio:

Gestión Operativa

  • Inventario Inteligente: Control de stock con alertas de agotado.
  • Catálogo CRUD: Editor visual de productos con galería de imágenes.
  • Order Management: Cambio de estados (Pagado, Enviado) en tiempo real.

Inteligencia de Negocio

  • Ventas Diarias: Gráficos de ingresos y pedidos completados.
  • Top Productos: Análisis de los artículos más deseados.
  • Data Export: Reportes detallados para contabilidad.

Infraestructura Escalable

  • PostgreSQL: Base de datos relacional sólida para integridad de datos financieros.
  • Global Edge Network: Despliegue en red de distribución para carga ultra rápida.
  • Asset Optimization: Compresión automática de imágenes subidas por el admin.
  • Vite Runtime: Proceso de construcción (build) optimizado para minimizar el peso del bundle.
  • SEO & Indexing: Arquitectura optimizada para motores de búsqueda y SSR dinámico.

Metodología y Desarrollo

  • Clean Code: Implementación de patrones de diseño React para facilitar la escalabilidad.
  • Scalability-First: Backend diseñado para soportar picos de tráfico durante temporadas altas.
  • Mantenimiento: Código autodocumentado con tipado estricto en el 100% del proyecto.
  • Unit Verification: Pruebas de flujo de integración para las pasarelas críticas.

Hoja de Ruta y Madurez

El proyecto ha alcanzado una fase de madurez Production-Ready. La arquitectura desacoplada permite integrar nuevas IAs, métodos de envío o mercados en cuestión de horas. Con una interfaz que prioriza la conversión y un backend robusto, MYB Hijos del Rey V2.0 es la herramienta definitiva para dominar el comercio digital moderno.
